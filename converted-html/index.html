Create web applications with less code

By drastically reducing the amount of code you need to write, Nooku helps you to create powerful web applications easily. Using the web technologies you know and love: PHP, HTML, CSS, and JavaScript

Getting started with Nooku is easy.

Nooku is an open source project for developers, integrators and hackers alike.

1. Grab Nooku Framework

Nooku Framework is still heavily in development, want a sneak peak?

Go ahead and grab the latest development version.

2. Install Nooku Server

Nooku Server is built on top of Vagrant to build our own configurable, lightweight, and portable virtual development environment.

Just download the latest stable version.

Nooku is different.

Our goal is to make it easier for developers to create more powerful products.

Intuitive API

Learn once, apply everywhere: the consistent API is easy to learn, and a joy to work with.

Out-of-the-box security

Nooku provides excellent input filtering, and protects against CSRF automatically.

Object Oriented design

The solid architecture makes use of the latest Object Oriented features in PHP, such as class autoloading,interfaces and visibility.

Unparalleled extensibility

Every part of Nooku, and of extensions powered by the framework, can be extended, re-used, and replaced.

Design pattern based

Nooku features a near zero-code MVC, an ORM with table behaviors, a powerful Chain of Command, an Object Factory with dependency injection, Lazy Loading, Mixins, Decorators, and more.

Extremely lightweight

Nooku has no dependencies on PECL extensions or PEAR libraries or any other libraries. Large, monolithic libraries are avoided in favor of optimized solutions.

Dynamic MVC

Nooku's dynamic MVC takes the work out of your hands. All you need to do is the code that is unique to your application.

User Interface

Nooku is not only about PHP code, it also comes with a powerful user interface based on the CSS Flexible Box Layout Module.
