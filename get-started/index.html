Start building with Nooku!

Follow these quick steps and start building Joomla extensions in minutes.

1. Create a composer.json file

Create a composer.json file in the root directory of your Joomla installation and require the nooku/nooku-framework package

{% highlight json %} { "require": { "nooku/nooku-framework": "dev-develop" }, "minimum-stability": "dev" } {% endhighlight %}

2. Run composer

Install by executing composer install.

{% highlight bash %} $ composer install {% endhighlight %}

3. Enable the plugin

Enable the plugin called "System - Nooku Framework" in Joomla.
