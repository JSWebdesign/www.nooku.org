What's Nooku?

For Joomla developers, integrators and hackers alike.

Nooku can be installed in Joomla as a plugin and lets you focus on the "business" logic of your extension.

By doing most of the repetitive work for you, boilerplate code is eliminated which greatly reduces the time to develop Joomla extensions.

The design pattern based architecture makes your extension to be more flexible, re-usable, and easily extensible.

Nooku also abstract differences between Joomla versions. Code it once, use it everywhere.

Who's using Nooku?

Each individually we turn Nooku's raw code into great products.
