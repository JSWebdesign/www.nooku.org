Getting started with Nooku is easy.

Nooku is an open source project for developers, integrators and hackers alike.

1. Grab our Framework

Nooku Framework is still heavily in development, want a sneak peak?

Go ahead and grab the latest development version.

2. Install our Server

Nooku Server is built on top of Vagrant to build our own configurable, lightweight, and portable virtual Nooku Framework development machine.

Just download the latest stable version.

3. Start developing

You are all set to start developing with Nooku, have fun!

How is Nooku different?

Nooku is not a product. We don’t try to solve user problems. Our goal is to make it easier for developers to create more powerful products, help them increase their productivity.

Object Oriented design

The solid architecture makes use of the latest Object Oriented features in PHP, such as class autoloading, interfaces and visibility.

Dedicated team

Driven by a professional team of developers, lead by Johan Janssens. Co-founder of Joomla and the Lead Architect of the Joomla 1.5 Framework.

Design pattern based

Nooku Framework features a near zero-code MVC, an ORM with table behaviors, a powerful Chain of Command, an Object Factory with dependency injection, Lazy Loading, Mixins, Decorators, and more.

Unparalleled extensibility

Every part of Nooku Framework, and of extensions powered by the framework, can be extended, re-used, and replaced.

Intuitive API

Learn once, apply everywhere: the consistent API is easy to learn, and a joy to work with.

Extremely lightweight

Nooku Framework has no dependencies on PECL extensions or PEAR libraries or any other libraries. Large, monolithic libraries are avoided in favor of optimized solutions.

Need help?

We let our code do the talking. We favor incubating our ideas into working code while trying to avoid long winded discussions. Our code always has the last word.

Guides

The aim of our guides is to introduce important concepts in Nooku for any competent PHP developer, along with some simple to follow examples, in order to get up to speed as quickly as possible.

Discussion Group

The Google group is the place to discuss the framework with fellow developers. It's also the place to be kept updates about framework progress. We are counting on you to help make Nooku the best framework out there!

Events

Nooku Jams are never your traditional conferences – they are more of an ad-hoc unconference born from the desire for people to share and learn in an open environment.
